== Description ==

Adds a 'Manage Patterns' child menu item under the WordPress 'Appearance' menu.

This plugin doesn't provide functionality for managing Patterns. Instead, it provides a shortcut to WordPress' built-in Pattern management.

No settings. Just enable the plugin, and the menu item will be added.
